# ERP CRM API Documentation

This directory contains the API documentation and supporting files for the ERP CRM Portal project.

## Contents

- `ERP_CRM_API.postman_collection.json` – Postman collection containing all API endpoints.
- Swagger API documentation is available at:

  https://erp-crm-portal-8p2u.onrender.com/api-docs

## API Modules

- Authentication
- Customers
- Products
- Categories
- Stock Management
- Challan Management
- Dashboard

## Backend API

https://erp-crm-portal-8p2u.onrender.com

## Frontend Application

https://erp-crm-portal-kappa.vercel.app
